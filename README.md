# ⚽ Football Player Price Prediction

A Machine Learning project that predicts the market value of football players based on their performance, physical attributes, technical skills, and overall ratings.

## 📌 Overview

The football transfer market involves millions of dollars in player transactions every season. Estimating a player's market value accurately can help clubs, scouts, analysts, and fans understand a player's worth.

This project uses Machine Learning techniques to analyze player statistics and predict their market value based on various performance-related features.

## 🚀 Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training and Evaluation
* Market Value Prediction
* Performance Visualization
* Prediction using trained Machine Learning model

## 📊 Dataset

The dataset contains information about football players including:

* Age
* Overall Rating
* Potential
* Pace
* Shooting
* Passing
* Dribbling
* Defending
* Physical Attributes
* Wage
* Nationality
* Club Information
* Market Value

## ⚙️ Machine Learning Workflow

### 1. Data Collection

Collected football player statistics and market values from the dataset.

### 2. Data Preprocessing

* Removed missing values
* Handled outliers
* Converted categorical variables
* Feature scaling

### 3. Exploratory Data Analysis

* Correlation Analysis
* Distribution Analysis
* Feature Importance Visualization

### 4. Model Training

The dataset was split into training and testing sets and trained using Machine Learning algorithms.

### 5. Model Evaluation

Performance was evaluated using regression metrics such as:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
